
# Comprehensive Debugging Setup (nvim-dap + nvim-dap-ui)

## Goal
Set up a complete debugging workflow in Neovim using `nvim-dap` and `nvim-dap-ui`.

## Language Priorities

### Primary
- JavaScript / TypeScript
- Python
- Go

### Secondary
- Rust
- C
- C#
- Bash

## Scope
- Install and configure `nvim-dap` and `nvim-dap-ui`.
- Set up language-specific debug adapters for each primary language.
- Define keymaps for common debug actions (continue, step over, step into, step out, toggle breakpoint, open UI).
- Integrate with the existing LSP and config refactoring effort.
